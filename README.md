
🔐 Password Strength Checker

A simple Python-based Password Strength Checker built using Google Colab. This project evaluates the strength of a password based on common security rules and provides suggestions to improve it.

📌 Features

-Checks password length (minimum 8 characters)

-Detects uppercase and lowercase letters

-Verifies presence of numbers

-Identifies special characters

-Classifies password as:

* Weak ❌

* Medium ⚠️

* Strong ✅

Provides suggestions for improving weak passwords

🛠️ Technologies Used

-Python

-Regular Expressions (re module)

-Google Colab

🚀 How It Works

-User enters a password.

-The program checks:

-Length

-Uppercase letters

-Lowercase letters

-Digits

-Special characters

A score is calculated based on these criteria.

The password is categorized and feedback is displayed.
